    Introduction #

    Welcome to the documentation for the Rock-Paper-Scissors game API. This API allows you to play against the computer and track the score.

    How to Play #

    1. Use the /play endpoint to play a round.
    2. Check the /score endpoint to see the current score.
    3. If you wish to restart the game, send a request to the /restart endpoint.

    Enjoy the game!

    Endpoints #

    POST /play #

    Play a round of Rock-Paper-Scissors.

    Request Body:

    {
  "choice": "rock"  // string (one of "rock", "paper", "scissors")
}

    Response:

    • user_choice (string): The choice you made.
    • api_choice (string): The choice made by the computer.
    • result (string): The result of the round, e.g., “You win!”

    GET /score #

    Retrieve the current score of the game.

    Response:

    • user (int): Number of rounds you’ve won.
    • api (int): Number of rounds the computer has won.
    • draws (int): Number of rounds that ended in a draw.

    PUT /restart #

    Restart the game, setting all scores back to zero.

    Response:

    • message (string): A confirmation message that the game has been reset.
    • score (object): The current (now reset) score.

    Note #

    If the server restarts, all scores will be reset to 0.
